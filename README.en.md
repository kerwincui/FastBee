# wumei-open

#### Description
物美智能设备开源项目，项目尽可能简单，同时保证功能的完整性，方便使用和二次开发。数据库使用mysql、后端spring boot、前端vue、移动端android、硬件交互基于mqtt协议使用EMQ服务器，硬件使用乐鑫的芯片，基于ESP-IDF框架。
